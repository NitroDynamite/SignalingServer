# Steps to Create Signal Server Files

1. **Create `signaling-server.js`:**
   - Open your code editor.
   - Create a new file named `server.js`.
   - Add your server code
   - example: https://github.com/paullouisageneau/libdatachannel/tree/master/examples/signaling-server-nodejs

2. Install Node.js: https://nodejs.org/en

3. **Create `package.json`:**
   - In the terminal, navigate to your project directory.
   - Run: npm init -y
 
   - This generates a basic `package.json` file.

4. **Install Dependencies:**
   - Run the following command to install required packages:
     npm install
